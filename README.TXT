## About

Krackdy is a debian-based remix, which includes kodi, some native-linux, open-source games that use gamepad, AEL, libretro cores and other emulators including xemu.

It does not include illegal roms.

The root password is "root"
The user name is "kodi" and the user's password is also "kodi"

Run
	sudo /sbin/installToHDD.sh /dev/sda
From the live environment to install to hard disk.

The image boots from both UEFI and legacy bios.
